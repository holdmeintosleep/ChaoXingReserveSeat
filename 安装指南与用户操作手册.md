## 概述

超星座位预约桌面应用是一款基于超星学习通平台的座位预约与签到管理工具。该应用提供了完整的座位预约、自动签到、配置管理等功能，帮助用户更高效地管理图书馆座位预约。

**主要功能：**
- 用户配置管理（支持多用户）
- 座位预约执行
- 自动签到功能
- 预约历史记录管理
- 定时任务管理

## 安装指南

### 方式一：使用已打包的应用程序（推荐）

1. **下载安装包**
   - 从项目发布页面下载最新版本的安装包

2. **解压安装包**
   - 将下载的压缩包解压到任意目录（建议路径不含中文）
   - 推荐路径：`C:\Program Files\ChaoxingReserveSeat`

3. **运行应用**
   - 双击解压目录中的 `ChaoxingReserveSeat.exe` 启动应用
   - 首次启动可能需要等待2-3秒加载时间

### 方式二：从源代码构建

#### 环境准备

1. 安装 Python 3.10+（推荐 3.12）
2. 安装 Node.js 16+

#### 安装依赖

```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 安装前端依赖
cd frontend
npm install
npm run build
cd ..
```

#### 运行开发模式

```bash
# 启动后端服务
python backend/app.py

# 启动前端开发服务器（新开终端）
cd frontend
npm run dev

# 或直接运行桌面应用
python desktop_app.py
```

#### 打包应用

```bash
# 使用 PyInstaller 打包
python -m PyInstaller build.spec --noconfirm
```

打包完成后，可执行文件位于 `dist/ChaoxingReserveSeat/` 目录下。

## 首次使用配置

### 1. 配置用户信息

应用首次启动后，需要添加用户配置：

1. 在"配置管理"页面点击"添加配置"
2. 填写以下信息：
   - **用户名**：超星学习通账号（学号或手机号）
   - **密码**：超星学习通密码
   - **预约时间**：选择需要预约的时间段
   - **房间ID**：图书馆房间ID
   - **座位号**：想要预约的座位号
   - **预约星期**：选择需要预约的星期

### 2. 获取房间ID

如果不知道房间ID，可以通过以下方式获取：

1. 打开超星学习通座位预约页面
2. 查看浏览器地址栏中的参数，格式类似 `id=xxxx`
3. 或者在配置页面点击"获取房间列表"按钮

### 3. 配置签到位置

在配置页面可以设置GPS位置信息，用于签到时的位置模拟：

- **纬度**：位置纬度
- **经度**：位置经度
- **地址**：位置描述

## 功能使用说明

### 配置管理

**页面位置**：左侧导航 → 配置管理

**功能说明**：

- **查看配置列表**：显示所有已添加的用户配置
- **添加配置**：点击"添加配置"按钮，填写用户信息
- **编辑配置**：点击配置项右侧的"编辑"按钮修改信息
- **删除配置**：点击配置项右侧的"删除"按钮移除配置
- **导入/导出配置**：支持将配置导出为JSON文件或从JSON文件导入

### 预约管理

**页面位置**：左侧导航 → 预约管理

**功能说明**：

- **执行预约**：选择配置后点击"执行预约"按钮
- **预约状态**：查看当前预约任务的执行状态
- **预约历史**：查看历史预约记录

### 签到管理

**页面位置**：左侧导航 → 签到管理

**功能说明**：

- **执行签到**：选择用户配置后点击"执行签到"按钮
- **签到状态**：查看当前签到任务的执行状态
- **批量签到**：一次性为所有用户执行签到

### 规则说明

**页面位置**：配置管理页面顶部

显示超星学习通座位预约的相关规则：

- **预约规则**：预约时间、时长、次数限制等
- **违规规则**：违约处理方式
- **使用流程**：预约 → 签到 → 使用 → 退座

## 常见问题

### Q1: 应用启动后白屏？

**可能原因**：后端服务未正常启动

**解决方案**：
1. 检查网络连接
2. 确认超星学习通平台可正常访问
3. 关闭应用后重新启动

### Q2: 预约失败？

**可能原因**：

1. **用户名/密码错误**：请检查配置中的账号信息
2. **房间ID错误**：请确认房间ID是否正确
3. **座位已被预约**：该座位在指定时间段已被他人预约
4. **网络问题**：网络连接不稳定

**解决方案**：
1. 核对配置信息
2. 尝试更换座位号
3. 检查网络连接

### Q3: 签到失败？

**可能原因**：

1. **预约ID不存在**：未找到有效的预约记录
2. **签到时间未到**：需要在预约开始时间后才能签到
3. **位置信息错误**：GPS位置与实际位置偏差过大

**解决方案**：
1. 先执行预约，获取预约ID后再签到
2. 确认签到时间是否在有效范围内
3. 调整GPS位置信息

### Q4: 如何备份配置？

**方法**：

1. 在配置管理页面点击"导出配置"按钮
2. 将导出的 `config.json` 文件保存到安全位置
3. 需要恢复时点击"导入配置"按钮选择备份文件

### Q5: 应用占用内存过高？

**说明**：

- 应用正常运行时内存占用约 100-200MB
- 如果内存占用持续升高，请重启应用
