# 常见报错排查清单

## ⚠️ 合规风险提示

**本工具仅用于Web自动化技术学习与研究目的**，严禁用于考试作弊等违规场景。

---

## 错误代码速查

### 错误1：浏览器启动失败

**错误信息**：
```
浏览器启动失败: ...
```

**可能原因**：
1. 未安装Chrome或Edge浏览器
2. 浏览器版本与驱动不兼容
3. 权限不足

**解决步骤**：
1. 检查是否安装了Chrome或Edge浏览器
2. 更新浏览器到最新版本
3. 以管理员身份运行程序

---

### 错误2：登录失败

**错误信息**：
```
登录失败，请检查账号密码
```

**可能原因**：
1. 账号密码错误
2. 网络问题
3. 验证码拦截
4. 账号被锁定

**解决步骤**：
1. 验证账号密码是否正确
2. 检查网络连接
3. 手动登录一次验证账号状态
4. 检查是否需要验证码

---

### 错误3：未识别到题目

**错误信息**：
```
未识别到题目，请检查页面是否正确
```

**可能原因**：
1. 页面结构变化
2. 未进入正确的答题页面
3. 页面加载不完全

**解决步骤**：
1. 确认当前页面是答题页面
2. 等待页面完全加载
3. 刷新页面重试
4. 更新工具版本

---

### 错误4：未找到答案

**错误信息**：
```
未找到答案，跳过此题
```

**可能原因**：
1. 搜题API配置错误
2. API密钥无效
3. 题目不在题库中
4. 网络问题

**解决步骤**：
1. 检查API配置是否正确
2. 验证API密钥是否有效
3. 尝试启用本地题库
4. 检查网络连接

---

### 错误5：填写答案失败

**错误信息**：
```
填写答案失败: ...
```

**可能原因**：
1. 页面元素定位失败
2. 选项格式不匹配
3. 页面加载延迟

**解决步骤**：
1. 等待页面完全加载
2. 刷新页面重试
3. 更新工具版本

---

### 错误6：提交失败

**错误信息**：
```
提交失败: ...
```

**可能原因**：
1. 提交按钮未找到
2. 确认弹窗未处理
3. 页面验证失败

**解决步骤**：
1. 手动检查提交按钮位置
2. 确保所有题目都已作答
3. 尝试手动提交

---

### 错误7：打包失败

**错误信息**：
```
错误: 打包失败
```

**可能原因**：
1. 依赖包未安装
2. PyInstaller版本不兼容
3. 路径包含中文或特殊字符

**解决步骤**：
1. 重新安装依赖包
2. 更新PyInstaller到最新版本
3. 将项目移动到纯英文路径

---

## 调试模式

### 启用详细日志

在 `config.json` 中添加：
```json
{
    "debug": true
}
```

### 查看浏览器控制台

程序运行时，按 F12 打开开发者工具，查看控制台输出。
