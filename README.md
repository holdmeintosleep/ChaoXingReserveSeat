# 超星学习通座位自动预约与签到系统

> 基于 Python 的超星学习通（学习通）座位预约和签到自动化工具，支持自动滑块验证和智能签到。

## 功能特性

### 自动预约模块
- ✅ 自动登录超星学习通账号
- ✅ 支持滑块验证码自动识别（基于 OpenCV 图像处理）
- ✅ 多用户、多时间段并发预约
- ✅ 预约成功后自动保存预约ID
- ✅ 定时抢座模式（22:00 自动启动）

### 自动签到模块
- ✅ 分段式签到 - 自动检测当前时间段并签到
- ✅ GPS 位置模拟 - 绕过位置检查机制
- ✅ 自动获取预约ID - 预约成功后自动保存，签到时自动读取
- ✅ 多用户支持 - 每个用户独立 session
- ✅ 手动填写模式 - 支持手动配置预约ID

### 技术特点
- 🐍 纯 Python 实现，无需浏览器驱动
- 🔐 AES 加密登录，安全可靠
- 📝 详细的日志记录
- 🔄 自动重试机制

## 快速开始

### 1. 安装依赖

```bash
pip install requests numpy opencv-python cryptography
```

### 2. 配置 config.json

```json
{
    "signin": {
        "enabled": true,
        "check_interval": 60,
        "location": {
            "latitude": 30.0,
            "longitude": 120.0,
            "address": "图书馆"
        }
    },
    "reserve": [
        {
            "username": "你的手机号",
            "password": "你的密码",
            "time": ["10:30", "14:15"],
            "roomid": "6913",
            "seatid": ["011"],
            "daysofweek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"]
        }
    ]
}
```

### 3. 运行程序

```bash
# 预约模式（22:00 自动抢座）
python main.py -m reserve

# 签到模式
python main.py -m signin

# Debug 模式（立即执行预约）
python main.py -m debug
```

## 配置说明

### 时间段配置
`time` 数组格式：`["开始时间", "结束时间"]`

示例：`["10:30", "14:15"]` 表示预约 10:30 - 14:15 时段

### 房间ID
| 房间ID | 名称 |
|--------|------|
| 8740 | 二楼208室-考研学习室 |
| 6913 | 其他图书馆 |

### 星期配置
`daysofweek` 支持以下值：
- `Monday`, `Tuesday`, `Wednesday`, `Thursday`, `Friday`, `Saturday`, `Sunday`

### GPS 位置
修改 `signin.location` 中的经纬度为实际图书馆坐标：
```json
"location": {
    "latitude": 30.5427,
    "longitude": 114.3628,
    "address": "武汉大学图书馆"
}
```

常用高校坐标：
- 武汉大学：30.5427, 114.3628
- 华中科技大学：30.5154, 114.4214
- 浙江大学：30.2634, 120.1236

## 预约ID管理

预约成功后，预约ID会自动保存到 `reservations.json` 文件：

```json
{
    "15367563648": {
        "2026-06-16": {
            "10:30-14:15": "184613190",
            "14:30-18:15": "184613191"
        }
    }
}
```

签到时会自动读取对应日期和时间段的预约ID，无需手动配置。

## 常见问题

### Q: 滑块验证码识别失败？
A: 确保已安装 OpenCV：`pip install opencv-python`

### Q: 预约失败显示"未登录"？
A: 检查用户名密码是否正确，确保账号未被封禁

### Q: 如何修改预约时间？
A: 修改 config.json 中 `reserve.time` 字段

### Q: 如何部署到服务器？
A: 推荐使用定时任务（如 crontab）配合 `-m debug` 模式
