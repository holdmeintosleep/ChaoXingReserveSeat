## 常见问题

### Q1: 我应该先学什么语言？

**建议：**
- 初学者首选 **Python**（语法简洁，应用广泛）
- 想做网页开发选 **JavaScript**
- 想做系统级开发选 **Go/Rust**

### Q2: 如何高效学习编程？

**核心原则：**
1. 每天坚持写代码（至少30分钟）
2. 动手实践比看视频更重要
3. 遇到问题先尝试自己解决
4. 参与开源项目或做个人项目

### Q3: 卡壳了怎么办？

**调试步骤：**
1. 仔细阅读错误信息
2. 在搜索引擎中搜索错误关键词
3. 简化代码，定位问题范围
4. 查看官方文档和社区问答

## 代码规范建议

```python
# 示例：Python代码规范
def calculate_average(numbers):
    """计算列表中数字的平均值"""
    if not numbers:
        return 0
    return sum(numbers) / len(numbers)
```

**规范要点：**
- 使用有意义的变量名
- 添加清晰的注释和文档字符串
- 遵循PEP8/Python风格指南
- 保持函数职责单一

**示例2：代码讲解**
```
用户：这段代码是什么意思？
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

导师：
这是斐波那契数列的递归实现：
- 基线条件：n ≤ 1时返回n
- 递归步骤：第n项 = 第n-1项 + 第n-2项
- 时间复杂度：O(2^n)，可以用动态规划优化
```
